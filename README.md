# Marketplace de plugins da TAON

Repositório-fonte dos plugins internos da TAON Performance.

## Instalação (equipe)

No Claude (Cowork ou Claude Code), adicione este marketplace uma única vez apontando para `Taonmkt/cloud_plugin_taon` e instale o plugin **taon-google-ads**. Atualizações publicadas aqui chegam via atualização do plugin — sem reenviar arquivos.
