# TAON Google Ads

Plugin da TAON Performance para análise de contas de Google Ads.

## Skills

- **analise-termos-google-ads** — Analisa relatórios de termos de pesquisa e palavras-chave (CSV exportado do Google Ads), classifica cada termo por intenção de busca, gera listas de palavras negativas seguras (regras oficiais de correspondência negativa), faz análise temporal com histórico de alterações, audita campanhas paralelas e landing page, e entrega planilha Excel com 5 abas (resumo, termos classificados, negativas prontas para colar, palavras-chave sugeridas por grupo e recomendações).

- ## Como usar

- Envie um CSV do Google Ads numa conversa do Claude e peça a análise. O skill dispara automaticamente e pergunta o contexto do negócio antes de classificar.

- ## Atualizações

- Este plugin é distribuído pelo marketplace da TAON. Para atualizar, use a opção de atualização de plugins nas configurações — a versão nova vem deste repositório.
